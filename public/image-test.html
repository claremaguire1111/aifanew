Troubleshooting Solutions

If the images from /images/social/ and /images/support/ are not loading, but regular images are, try these solutions:

Solution 1: Update AIFAImage component

If you are using the AIFAImage component, it will now use absolute URLs for these folders in production.

Solution 2: Manually update image paths

Change image paths from:

<img src="/images/social/AIFA1.png" />

To:

<img src="https://aifilmacademy.io/images/social/AIFA1.png" />

Solution 3: Use Next.js Image component

Replace standard img tags with Next.js Image component:

import Image from 'next/image';
<Image src="/images/social/AIFA1.png" width={500} height={300} />
